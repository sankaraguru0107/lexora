# LEXORA — AI-Powered Legal Assistant

A full-stack React app for Indian law assistance with AI chat, voice, and document generation.

## Setup
```
npm install
npm start
```
Open http://localhost:3000

## Build for Production
```
npm run build
serve -s build
```

## Features
- AI Chat with Azure OpenAI 
- Voice interaction (Web Speech API)
- 3 role dashboards: User, Advocate, Admin
- Legal Topics browser (8 Indian law categories)
- Document Generator (AI-powered)
- Consultation history
- Dark purple UI matching Lexora screenshots
